Access Modes

Mode Description Restrictions
v Voice. This makes the user able to speak in +m/+M channels. May be set by +hoaq
h Half-Op. Gives access to some of the channel operator privledges. May be set by +oaq
o Channel Op. Gives access to all the channel operator privledges. May be set by +oaq
a Channel Admin. Same privledges as +o but can't be kicked by +ho. May be set by +q
q Channel Owner. Can't be kicked by anyone below. Normal users can't set this

List Modes

Mode Description Restrictions
b Ban. Prevents a user from joining the channel. Requires +h or higher
e Ban exception. When a user is banned (due to +b) and they are on this +e list then they may still join the channel. Requires +h or higher
I Invite exception. When the channel is +i (invite only) then people on this list may still join the channel. Requires +h or higher

See Exteneded Bans for more advanced list mode usage.


Channel Modes

Mode Description Restrictions
c Colored messages are not permitted. Requires +o or higher
C CTCP's are not permitted. Requires +o or higher
D Delay JOIN messages until that person speaks. (Chanops, IRCOps, & Services are exempt) Requires +o or higher
f Flood protection. (See here for more details) Requires +o or higher
G Enable word filters. Requires +o or higher
i Requires users to be /INVITE'd to join the channel. Requires +o or higher
k Requires a channel key in order to join. Requires +h or higher
K KNOCK command is not allowed. Requires +o or higher
L Channel link if the +l limit is reached. Requires +h or higher
l Limit the amount of users that may be in the channel. Requires +o or higher
m Moderated channel. Requires +v or higher to speak. Requires +o or higher
M Only registered nicks and +v can chat. Requires +o or higher
N /NICK is not permitted. Requires +o or higer
n No external messages. Requires +h or higher
O IRC operator only channel. IRCOp-only
P Preserve channel settings if the channel becomes empty. IRCOp-only
p Private channel. Requires +o or higher
Q /KICK is not permitted. (Kicking via services still works) Requires +o or higher
R Only registered users may join the channel. Requires +o or higher
r Channel is registered via services. Set by services
s Hide channel from LIST and WHOIS replies. Requires +o or higher
S Strip color from messages. Requires +o or higher
T /NOTICE is not permitted. Requires +o or higher
t Do not allow regular users to change the topic. Requires +h or higher
V /INVITE is not permitted. Requires +o or higher
z Only clients which are connected through SSL/TLS may join the channel. Requires +o or higher
Z Indicates that only people who are using SSL/TLS are on the channel. Set by server
